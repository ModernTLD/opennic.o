Reserved Domains

Certain domains are set aside (registered to "ModernTLD"), for specific policy and technical purposes.

Example Domains

Following the guidance of specifications including RFC 6761, the following domains have been set aside for documentation purposes. These domains may be used as illustrative examples in documents without prior coordination with ModernTLD or any individual domain holder. They will never be available for registration or transfer.

  1. example.o
  2. test.o
Localhost Domain

Following the guidance of specifications including RFC 6761, the domain localhost.o has been set aside for testing and documentation purposes. This domain has an A record pointing to 127.0.0.1 and an AAAA record pointing to ::1. This domain is intentionally set to the loopback address, and will never be available for registration or transfer.

Organizational Domains

The following domains serve various purposes within the ModernTLD organization:

  • register.o -- Domain Registration
  • reg.o -- Redirects to register.o
  • tld.o -- Legacy domain registration page, Redirects to register.o
  • opennic.o -- (This website) ModernTLD policies/info page.
  • nic.o -- ModernTLD NIC Information.
  • modern.o -- Future ModernTLD homepage. Redirects to opennic.o
  • search.o -- Reserved.
  • mail.o -- Reserved.
  • ns1.o -- Legacy; Reserved.
  • ns2.o -- Legacy; Reserved.
  • ns3.o -- Legacy; Reserved.
  • ns4.o -- Legacy; Reserved.
